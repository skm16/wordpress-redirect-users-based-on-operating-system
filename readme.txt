Allows admins to create custom redirects based the user\'s operating system for posts and pages.

== Installation ==
1. Download and install plugin
2. Activate plugin
3. Add custom redirects via the page or post editor
4. Leave fields blank if no redirect is desired

== Frequently Asked Questions ==
Which operating systems does this plugin support?
Currently this plugin detects most Mac, Windows and Linux operating systems.

Does the plugin support custom post types?
No, not currently.
